------------------------------------------------------------------------
Este programa tiene como propósito encontrar la menor cantidad de saltos
que debe dar un caballo en un tablero de ajedrez para llegar desde una
casilla de origen dada hasta una casilla de destino dada.
------------------------------------------------------------------------

PROJECT TITLE: Caballo
PURPOSE OF PROJECT: encontrar la menor cantidad de pasos de un caballo
en un tablero de ajedrez para ir de una casilla a otra.
VERSION or DATE:8/8/2017
AUTHORS: Lida Heliana Arenas Mogollón
USER INSTRUCTIONS: El usuario ingresa una coordenada de origen y una
coordenada de destino.
